스테이지 배경 이미지 폴더
=========================

이 폴더(Assets/_Project/Resources/Backgrounds/)에 세로 배경 이미지를 넣으면
TapBattleGame이 자동으로 불러와 스테이지마다 순서대로 바꿔 씁니다.

규칙
----
- 파일 이름의 "정렬 순서"가 곧 스테이지 순서입니다.
  예: bg_01.png(스테이지1), bg_02.png(스테이지2), bg_03.png ...
- 스테이지 수가 배경 수보다 많아지면 처음부터 순환합니다.
- 권장 해상도: 세로 1080x1920 (또는 비슷한 9:16 비율).
- PNG/JPG 모두 가능. 런타임에 Texture2D로 읽어 스프라이트로 만들기 때문에
  Import 설정(Sprite/Default)과 무관하게 동작합니다.

지금 받은 두 이미지
-------------------
- 초록 숲  -> bg_01.png 로 저장
- 얼음 숲  -> bg_02.png 로 저장
